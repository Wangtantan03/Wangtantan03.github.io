# python基础

# python range（）函数用法

range()函数可用于创建一个整数列表，一般用在for循环中

# 语法

range(start, end, step)

  • start: 从start开始, 默认从0开始， range(5)等价于range(0,5)
  • end: 到end结束,但不包括end; range(1,5)是 [1, 2, 3, 4]没有5
  • step: 步长，默认步长为1；range(1,5)等价于range(1,5,1)

# 实例

>>> for i in range(1,10,2):
       print("i=",i)
i = 1
i = 3
i = 5
i = 7
i = 9

>>> for i in range(4):
       print("i=",i

i = 0
i = 1
i = 2
i = 3

# python len() 函数用法

len()函数返回对象，如字符、列表、元组等的长度

# 语法

len(l) , 其中l为对象

# 实例

>>> str = "wangtantan"
>>> len(str) # 字符串长度
10
>>> l = [1,2,3,4,5]
>>> len(l)  # 列表元素个数
5
